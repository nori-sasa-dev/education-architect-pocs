以下のコーディング規約に従ってください：

- **コメント**: 日本語で記述
- **デモモードパターン**: APIキー未設定時は `DEMO_RESPONSES` から固定応答を返す
- **エージェントクラス**: `XxxAgent(api_key=None)` → `respond(messages, turn)` → `extract_xxx(text)`
- **抽出マーカー**: `[MARKER_NAME]` + ` ```json ``` ` ブロック
- **環境変数**: `.env` から読み込み。`.env.example` を必ず用意
- **仮想環境**: 各アプリ独立の venv

## 注意事項

- 大きな変更を行う前に、笹川さんに確認を取ること
- 新しいライブラリを追加する場合は `requirements.txt` を更新すること
- `.env` ファイルや APIキーをコードにハードコードしないこと
- 既存の動作を壊さないよう、変更箇所を最小限にすること
- わからないことがあれば、推測で実装せず笹川さんに確認すること
