# セッション記録 #07｜2026-03-23

## セッション概要

新POC「FamilyHub」の企画・設計・Phase1実装まで一気通貫で完了。
子どもの学校・保育園・習い事からの情報が複数アプリに分散している課題を解決するため、
コドモンのメール通知をGoogle Apps Script + Claude APIで自動解析し、Googleカレンダーに登録する仕組みを構築した。

## 2. 決定事項

| # | 決定内容 | 理由・背景 |
|---|---------|-----------|
| 1 | GAS + Claude APIのGmailトリガー方式を採用 | 各アプリのAPI不要・サーバー不要・Googleカレンダーと直結できる |
| 2 | 抽出対象を6種に限定 | 休園・休校・早帰り・給食なし・イベント・提出物・面談のみ |
| 3 | priorityで「high」と「normal」を分類 | 休園・早帰り・給食なしは当日に影響するため前日＋1時間前の二重リマインダー |
| 4 | 処理済みラベルで重複防止 | 同じメールを二重登録しないよう「コドモン処理済み」ラベルを自動付与 |
| 5 | 送信元ドメインは `codmon.com`（要注意） | `codomon.com` ではない。今後コドモン関連で検索するときに注意 |
